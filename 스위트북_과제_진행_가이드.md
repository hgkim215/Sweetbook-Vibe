## 1. 목표 레벨 결정

이번 과제는 Lv1, Lv2, Lv3로 나뉜다. 전부 구현할 필요는 없지만, 완성도 기준으로 판단하면 아래 전략이 가장 현실적이다.

### 권장 목표

**Lv2 완성 + Lv3 최소 구현**

- Lv1만으로도 평가 대상이지만, 풀스택 개발자 과제이므로 콘텐츠 CRUD만 있으면 인상이 약할 수 있다.
- Lv2 주문 기능은 비즈니스 로직을 보여주기 좋고 구현 난이도도 적당하다.
- Lv3는 거창한 PDF 생성이 아니라 주문 데이터를 JSON 또는 ZIP으로 내보내면 되므로, 최소 구현해두면 데이터 모델링 능력을 보여줄 수 있다.

### 각 레벨별 구현 범위

| 레벨 | 구현해야 할 것 | 최소 완성 기준 |
|---|---|---|
| Lv1 | 콘텐츠 서비스 핵심 기능 | 콘텐츠 생성, 조회, 수정, 삭제 또는 주요 플로우 동작 |
| Lv2 | 책 주문 기능 | 특정 콘텐츠 묶음을 책으로 주문하고, 주문 상태를 관리 |
| Lv3 | 주문 데이터 익스포트 | 주문 1건에 필요한 콘텐츠와 메타데이터를 JSON 또는 ZIP으로 다운로드 |

## 2. 서비스 아이디어 선택

과제에서 요구하는 구조는 “콘텐츠 서비스가 본체, 책은 부가 기능”이다. 따라서 처음부터 책 제작 서비스로 접근하기보다, 사용자가 계속 콘텐츠를 쌓는 서비스를 잡는 것이 좋다.

### 추천 아이디어

**라이프로그 기반 여행 기록 서비스**

- 사용자가 여행별로 날짜, 장소, 사진 설명, 감정, 에피소드를 기록한다.
- 여행 기록을 타임라인 또는 카드 형태로 볼 수 있다.
- 특정 여행 또는 기간을 선택해 “여행 책 만들기” 주문을 생성한다.
- 주문에는 제목, 수령자명, 요청사항, 포함할 기록 목록, 상태가 저장된다.
- 주문 데이터는 파트너 전달용 JSON으로 익스포트한다.

### 이 아이디어가 과제에 맞는 이유

- 콘텐츠가 본체다: 여행 기록 작성, 조회, 편집이 핵심 서비스다.
- 책 기능이 자연스럽다: 여행 기록을 모아 포토북이나 여행 에세이북으로 만들 수 있다.
- 더미 데이터 만들기 쉽다: 제주 여행, 도쿄 여행, 부산 여행 등 샘플을 바로 넣을 수 있다.
- 면접 설명이 쉽다: 사용자의 기록 자산을 책이라는 물리 결과물로 확장하는 구조라고 설명할 수 있다.

## 3. 기술 스택 결정

가장 중요한 기준은 빠르게 완성하고, Docker로 안정적으로 실행되고, 면접에서 설명 가능한 것이다.

### 권장 스택

- 프론트엔드: React + Vite
- 백엔드: Node.js + Express
- DB: SQLite
- 스타일링: CSS 또는 Tailwind CSS
- 실행 환경: Docker Compose

### 선택 이유

- React + Vite는 화면 구현 속도가 빠르다.
- Express는 API 설계와 주문 상태 관리 구현이 단순하다.
- SQLite는 별도 DB 서버 없이 파일 기반으로 동작해 Docker 제출에 적합하다.
- Docker Compose에서 프론트, 백엔드, DB 파일을 간단히 묶을 수 있다.

- 여행 기록 생성

   - 선택한 기록 확인

5. 주문 목록/상세
   - 주문 상태 확인

## 7. API 설계

프론트와 백엔드 분리를 보여주기 위해 REST API로 단순하게 구성한다.

| Method | Path | 설명 |
|---|---|---|
| GET | `/api/entries` | 여행 기록 목록 |
| GET | `/api/entries/:id` | 여행 기록 상세 |
| POST | `/api/entries` | 여행 기록 생성 |
| PUT | `/api/entries/:id` | 여행 기록 수정 |
| DELETE | `/api/entries/:id` | 여행 기록 삭제 |
| GET | `/api/orders` | 주문 목록 |
| GET | `/api/orders/:id` | 주문 상세 |
| POST | `/api/orders` | 주문 생성 |
| PATCH | `/api/orders/:id/status` | 주문 상태 변경 |
| GET | `/api/orders/:id/export` | 주문 데이터 JSON 익스포트 |
