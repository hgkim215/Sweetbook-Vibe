# GrowthBook 제품 명세

## 1. 제품 정의

GrowthBook은 흩어진 학습, 프로젝트, 회고, 실패, 개선 기록을 `성장기록`으로 정리하고, 선택한 기록을 `성장기록집`으로 주문할 수 있게 하는 콘텐츠 서비스다.

서비스의 본체는 성장기록 관리다. 책 주문은 쌓인 성장기록을 소장 가능한 성장기록집으로 전환하는 부가 기능이다.

## 2. 대상 사용자

- 취준생
- 주니어 직장인
- 대학생/학습자
- 공통점: 자신의 성장 과정을 눈으로 확인하고, 필요할 때 회고나 포트폴리오 자료로 꺼내고 싶은 사람

## 3. 완성 레벨

| 레벨 | 구현 범위 | 완료 기준 |
|---|---|---|
| Lv1 | 성장기록 콘텐츠 서비스 | 더미 데이터 조회, 생성, 상세, 수정, 삭제가 동작한다. |
| Lv2 | 성장기록집 주문 기능 | 선택한 성장기록으로 주문을 만들고, 주문 상세와 상태 변경이 동작한다. |
| Lv3 | 주문 데이터 익스포트 | 주문 1건의 주문 정보, 챕터, 포함 기록, 메타데이터를 JSON으로 다운로드한다. |

## 4. 핵심 기능

### 성장기록

- 성장기록 목록 조회
- 성장기록 상세 조회
- 성장기록 생성
- 성장기록 수정
- 성장기록 삭제
- 카테고리, 태그, 날짜, 성과, 배운 점, 다음 행동 기록

### AI 보조 정리자

- 선택한 성장기록을 바탕으로 챕터 구성을 제안한다.
- 각 챕터의 짧은 요약과 누락 질문을 제안한다.
- `.env`에 `OPENAI_API_KEY`가 있으면 OpenAI API를 사용할 수 있다.
- 키가 없거나 API 호출이 실패하면 규칙 기반 Mock 제안으로 fallback한다.
- AI는 내용을 대신 꾸며내지 않고, 사용자가 쓴 기록을 정리하는 보조 역할만 한다.

### 성장기록집 주문

- 사용자가 성장기록 여러 개를 선택한다.
- 자동 제안된 챕터 구성을 확인한다.
- 사용자가 책 제목, 작성자명, 요청사항, 챕터 제목, 포함 기록을 수정한다.
- 주문 상태를 관리한다.
  - `pending`
  - `processing`
  - `completed`
  - `cancelled`

### 주문 데이터 익스포트

- 주문 상세 화면에서 JSON 익스포트를 다운로드한다.
- JSON에는 주문 정보, 챕터 구성, 포함 성장기록, 익스포트 메타데이터가 포함된다.
- 출력용 PDF는 만들지 않는다.

## 5. 데이터 개념

### 성장기록

- 제목
- 카테고리: 프로젝트, 학습, 실패, 개선, 회고, 성과
- 날짜
- 요약
- 본문
- 배운 점
- 결과
- 다음 행동
- 태그

### 챕터 제안

- 챕터 제목
- 챕터 요약
- 포함 성장기록 ID 목록
- 누락 질문

### 성장기록집 주문

- 책 제목
- 작성자명
- 요청사항
- 상태
- 선택된 성장기록
- 챕터 구성
- 생성일/수정일

## 6. 제외 범위

- 회원가입/로그인
- 결제
- 배송
- 실제 `api.sweetbook.com` 호출
- 인쇄용 PDF 생성
- 채용 평가 자동화
- 의료, 상담, 법률 조언

## 7. 성공 기준

- 로그인 없이 더미 데이터로 핵심 플로우를 바로 확인할 수 있다.
- Lv1-Lv3가 하나의 사용자 여정으로 자연스럽게 이어진다.
- OpenAI 키가 없어도 앱이 정상 동작한다.
- Docker Compose 명령 1회로 실행된다.
- README만 보고 심사자가 실행하고 평가할 수 있다.
